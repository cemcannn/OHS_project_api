# Excel Veri Şablonları

Bu klasörde örnek Excel dosyaları bulunmaktadır.

## Kullanım

1. Aşağıdaki dosya isimlerinden birini kullanın:
   - `Veri.xlsx` - Kaza verileri için
   - `Veri Yevmiye.xlsx` - Aylık istatistik verileri için

2. Dosyaları hazırladıktan sonra bu klasöre yerleştirin

3. Sistem otomatik olarak dosyaları işleyecek ve `Processed/` klasörüne taşıyacaktır

## Dikkat

- Dosya isimleri tam olarak yukarıdaki gibi olmalıdır (büyük-küçük harf duyarlı)
- İlk satır başlık satırı olmalıdır
- Tarih formatı: DD.MM.YYYY (örn: 15.01.2024)
- Ay isimleri Türkçe olmalıdır (Ocak, Şubat, vb.)

Detaylı bilgi için ana klasördeki README.md dosyasına bakınız.
